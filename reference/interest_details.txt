# Interest Payment Details

## Workflow

At the top of the screen, workflow information to the interest payment is displayed for reference for the user.

  * Agreement Name
  * Principal Name
  * Counterparty Name
  * Margin Type
  * Amount
  * Agreed
  * Preference
  * Workflow State
  * Calculation Timestamp



## Calculation Status

This section will display the status of the interest payment calculation. If the calculation is not successful, then the system will return an explanation of what occurred.

## Movement Information

This section will display the information about the movement that the interest payment calculation is based upon.

  * Status
  * Settlement Date
  * Type
  * Direction
  * Instrument Ref
  * Ccy
  * Notional
  * Market Value
  * Physical Settlement



## Interest Information

This section will display information for the corresponding Interest Payment.

  * Currency
  * Held Index
  * Held Spread
  * Held Calculation Method
  * Held Day Count Convention
  * Held Period Start
  * Posted Index
  * Posted Spread
  * Posted Calculation Method
  * Posted Day Count Convention
  * Posted Period Start
  * Period End
  * Interest Due To (Principal/Counterparty)
